Ports

  • UDP (unreliable) is connectionless and smaller, TCP (reliable) has acknowledgment and is more data
  • File sharing
    • FTP - TCP 20, 21
      • Share files with users in a LAN or WAN unencrypted
    • SFTP - TCP 22
      • encrypted FTP
    • SMB and CIFS - TCP 445
      • Server Message Block
        • provides file sharing, network browsing, and printing commonly used in windows
      • Common Internet File System
        • open implementation used on Linux and Mac
  • Command line access
    • Telnet - TCP 23
      • unencrypted remote command line access
    • SSH - TCP 22
      • Encrypted remote command line access
      • SFTP - Encrypted FTP, same port
  • E-mail
    • SMTP - TCP 25
      • sends outgoing mail
    • POP3 - TCP 110
      • Downloads incoming mail from server
    • IMAP - TCP 143
      • synchronizes incoming mail from a server
  • Webpages
    • HTTP - TCP 80
      • transmits webpages but unsecured
    • HTTPS - TCP 443
      • transmit encrypted webpages with SSL or TLS
  • DNS - TCP/UDP port 53
    • Provides name to IP address resolution for host
  • DHCP - UDP 67, 68
    • provides configurations to clients in an IP network
  • SNMP - UDP 161, 162
    • Query, configure, and monitor host in a LAN
    • SNMPv3 encrypts communication
  • LDAP - TCP/UDP port 389
    • used in domain based network environments to facilitate system and user management
  • RDP - TCP 3389
    • Used to securely remote access a Windows desktop
